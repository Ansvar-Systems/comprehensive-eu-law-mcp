# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Comprehensive EU Law MCP, please report it through [GitHub Security Advisories](https://github.com/Ansvar-Systems/comprehensive-eu-law-mcp/security/advisories/new).

Do NOT create a public issue for security vulnerabilities.

## Scope

This MCP server provides advisory legal intelligence only. It does not:
- Connect to or modify any legal systems
- Access cloud provider APIs
- Store customer data
- Process authentication credentials

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
